# API Tap - 点击接口捕捉器 🎯

API Tap - 点击接口捕捉器 | Chrome Extension (Manifest V3) 捕捉浏览器点击后所有 Fetch/XHR 接口，按会话展示

## 功能特性

- **一键捕捉** — 点击页面任意元素，自动记录之后 10 秒内发起的全部网络请求
- **会话化管理** — 每次点击为一个 Session，独立展示请求列表，展开/折叠自如
- **实时刷新** — 活跃会话每 2 秒自动更新请求列表，已完成会话保持 DOM 稳定
- **接口类型识别** — 自动识别 XHR、Fetch、Document、Script、CSS、Image 等资源类型
- **URL 精简显示** — 默认截断过长 URL，支持一键切换显示完整 URL（含查询参数）
- **一键复制** — 每个请求旁带复制按钮，点击即复制 URL
- **导出报告** — 一键导出全部接口为文本文件（.txt），含时间戳与统计信息
- **智能防抖** — 同一元素 500ms 内不重复记录，避免误触发
- **数据持久化** — 使用 `chrome.storage.session` 存储，浏览器标签页关闭后自动释放

## 安装方式

### 从源码加载（开发者模式）

1. 克隆本仓库到本地：

   ```bash
   git clone https://github.com/<your-username>/api-tap.git
   ```

2. 打开 Chrome 浏览器，进入 `chrome://extensions/`

3. 开启右上角的 **「开发者模式」**

4. 点击左上角 **「加载已解压的扩展程序」**

5. 选择克隆下来的 `api-tap` 文件夹

6. 扩展图标出现在工具栏中，点击即可打开捕捉面板

## 使用方法

1. **浏览任意网页**，扩展默认处于监听状态
2. **点击页面上任意元素**（按钮、链接、图片等），扩展会记录此次点击
3. **点击扩展图标**，打开弹出面板查看捕捉结果
4. 在面板中：
   - ▶ 展开/折叠某个会话的请求列表
   - 📋 复制某个请求的 URL
   - 📥 导出全部数据为文本文件
   - ✕ 清空所有记录

## 技术架构

```
┌──────────────────┐     CLICK_EVENT     ┌─────────────────────┐
│   content.js     │ ──────────────────► │   background.js     │
│  (页面内容脚本)   │                     │  (Service Worker)   │
│  捕获点击事件     │                     │  管理会话与请求      │
└──────────────────┘                     └────────┬────────────┘
                                                  │
                                     WebRequest API│监听网络请求
                                                  │
                                                  ▼
┌──────────────────┐     GET_SESSIONS    ┌─────────────────────┐
│   popup.js/css   │ ◄────────────────── │   浏览器网络请求     │
│  (弹出面板)       │                     │                     │
│  展示与导出       │                     └─────────────────────┘
└──────────────────┘
```

### 文件结构

```
api-tap/
├── manifest.json          # 扩展清单（Manifest V3）
├── background.js          # Service Worker — 核心逻辑
├── content.js             # 内容脚本 — 捕获点击事件
├── popup.html             # 弹出面板 HTML
├── popup.js               # 弹出面板逻辑
├── popup.css              # 弹出面板样式
├── icons/                 # 扩展图标
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
├── .gitignore
└── README.md
```

## 权限说明

| 权限 | 用途 |
|------|------|
| `webRequest` | 监听页面发出的所有网络请求 |
| `storage` | 在 Session 级别存储会话和请求数据 |
| `activeTab` | 获取当前标签页信息 |
| `<all_urls>` | 允许在所有网站上运行内容脚本和监听请求 |

## 开发相关

### 配置项

在 `background.js` 顶部可调整捕捉窗口时长：

```js
const COLLECTION_WINDOW_MS = 5000; // 默认 5 秒
```

### 构建与打包

无需构建工具，直接以源码加载即可使用。

- 压缩图标文件（当前 PNG 较大）
- 添加隐私政策文档
- 完善扩展商店截图
