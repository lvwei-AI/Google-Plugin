**API Tap** 是一款 Chrome 浏览器扩展（Manifest V3），用于**捕捉页面点击后触发的所有 API 接口**（仅 Fetch / XHR），按点击会话（Session）有序展示，帮助开发者快速定位每次点击行为调用了哪些后端接口。

## 功能特性

- **🎯 精准捕捉** — 点击页面任意元素，自动捕获后续触发的所有 **Fetch 和 XHR 请求**（忽略图片、脚本、CSS 等资源）
- **📦 会话化管理** — 每次点击为一个 Session，独立展示请求列表，展开/折叠自如
- **⏱ 智能防抖** — 每次捕获到新接口自动重置计时器，**连续 10 秒无新接口**才认为捕捉完成，慢接口不会遗漏
- **🔄 实时刷新** — 活跃会话每 2 秒自动更新请求列表，已完成会话保持 DOM 稳定
- **🔍 URL 精简显示** — 默认截断过长 URL，支持一键切换显示完整 URL（含查询参数）
- **📋 一键复制** — 每个请求旁带复制按钮，点击即复制 URL
- **📊 导出 CSV** — 一键导出全部数据为 CSV 文件，Excel / WPS 直接打开
- **🛡 全局防抖** — 1 秒内全局只记录一次点击，避免快速连点刷出空会话
- **💾 数据持久化** — 使用 `chrome.storage.session` 存储，浏览器标签页关闭后自动释放

## 安装方式

### 从源码加载（开发者模式）

1. 克隆本仓库到本地：

   ```bash
   git clone https://github.com/lvwei-AI/Google-Plugin.git
   ```

2. 打开 Chrome 浏览器，进入 `chrome://extensions/`

3. 开启右上角的 **「开发者模式」**

4. 点击左上角 **「加载已解压的扩展程序」**

5. 选择 `Google-Plugin` 文件夹

6. 扩展图标出现在工具栏中，点击即可打开捕捉面板

## 使用方法

1. **浏览任意网页**，扩展默认处于监听状态
2. **点击页面上任意元素**（按钮、链接等），扩展会记录此次点击及后续 API 请求
3. **点击扩展图标 🎯**，打开弹出面板查看捕捉结果
4. 在面板中：
   - ▶ 展开/折叠某个会话的请求列表
   - 📋 复制某个请求的 URL
   - 📥 CSV 导出全部数据
   - ✕ 清空所有记录

## 权限说明

| 权限 | 用途 |
|------|------|
| `webRequest` | 监听页面发出的 Fetch / XHR 网络请求 |
| `storage` | 在 Session 级别存储会话和请求数据 |
| `activeTab` | 获取当前标签页信息 |
| `<all_urls>` | 允许在所有网站上运行内容脚本和监听请求 |

## 开发相关

### 配置项

在 `background.js` 顶部可调整接口防抖时长：

```js
const REQUEST_DEBOUNCE_MS = 10000; // 默认 10 秒无新接口则认为本次捕捉完成
```
